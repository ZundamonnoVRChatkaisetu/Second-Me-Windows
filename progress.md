## 実行方法
1. 新UI版の起動:
   ```
   start-new-ui.bat
   ```

2. 元のUIで起動（チャットのみ）:
   ```
   start-all-in-one.bat
   ```
   
3. CORS解決版を起動:
   ```
   start-with-cors.bat
   ```
